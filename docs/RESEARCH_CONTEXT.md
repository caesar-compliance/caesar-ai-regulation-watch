# Research Context — Caesar AI Regulation Watch

## Role

Monitoring AI regulation, official guidance, EU AI Act implementation, Norway implementation, and related sources.

## What it should become

A public regulatory timeline and source registry that maps changes to controls and evidence.

## Product type

- data repo
- source registry
- timeline
- changelog
- static site
- future alerting service

## Main users

- lawyers
- compliance teams
- consultants
- SMEs using AI systems

## Main use cases

- monitor EU AI Act sources
- monitor EU AI Office guidance
- monitor Norway implementation and Datatilsynet updates
- map regulatory changes to controls/evidence
- prepare client memos

## MVP

- source registry
- regulatory change schema
- timeline file
- affected controls mapping
- sample change reports

## Future paid use

- regulatory inbox
- client-specific impact memos
- deadlines and reminders
- task generation
- report exports

## Promotion

AI regulatory changes mapped to controls and evidence.
